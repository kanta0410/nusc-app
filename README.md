# NUSC 部活動管理システム

部活動の活動登録・欠席連絡を管理するWebアプリケーションです。

## 技術スタック
- Next.js 14 (App Router) + TypeScript
- Tailwind CSS + Lucide React

## セットアップ

```bash
npm install
npm run dev
```
→ http://localhost:3000 を開く

## 機能
### 管理者（青テーマ）
- 生徒の一覧・新規登録
- 活動日の登録・削除
- 欠席者の確認

### 生徒（緑テーマ）
- 活動カレンダーの閲覧
- 欠席連絡の送信
- 連絡済み確認

## データについて
インメモリストア使用。サーバー再起動でリセットされます。
本番環境では src/lib/db.ts をDBに差し替えてください。
